Mes projet

Je développe plateforme numérique ète et captivante, conçue pour transformer l'expérience souvent fastidieuse de la recherche de logements. Mon objectif est de fluidifier l'intégralité du processus, de l'idée initiale à la conclusion du bail, grâce à une solution innovante et intuitive.

Objectif du projet

Créer une plateforme100 % fiable, accessible et sécurisée permettant à tout utilisateur de trouver un logement réel, vérifié et conforme, tout en éliminant les arnaques et les faux intermédiaires

Technologies utilisées (prévisionnel)

  • Backend : Laravel / PHP
  • Frontend : Vue.js ou React
  • Mobile : Flutter ou React Native
  • Base de données : MySQL
  • Cloud Storage : Firebase / AWS / OVH
  • Paiements : TMoney, Flooz, PayDunya, Flutterwave
